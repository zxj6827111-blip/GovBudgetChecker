## 3. AI客户端调用方法错误

**文件**：services/ai_findings.py
**函数**：_call_ai_client
**行号**：173-193

**BUG描述**：代码尝试调用self.ai_client.chat_completion，但现有AI客户端可能没有这个方法，导致AI调用失败。

**触发条件**：执行AI分析时
**影响范围**：所有AI分析功能
**严重程度**：高

**代码问题**：
```python
async def _call_ai_client(self, prompt: str) -> str:
    try:
        # 使用现有的AI客户端接口
        if hasattr(self.ai_client, 'chat_completion'):
            response = await asyncio.wait_for(
                self.ai_client.chat_completion(
                    messages=[{"role": "user", "content": prompt}],
                    temperature=self.config.ai_temperature,
                    max_tokens=4000
                ),
                timeout=self.config.ai_timeout
            )
            return response.get("content", "")
        else:
            # 如果没有异步方法，使用同步方法
            import asyncio
            response = await asyncio.get_event_loop().run_in_executor(
                None, self._sync_call_ai, prompt
            )
            return response
    except Exception as e:
        logger.error(f"AI客户端调用失败: {e}")
        raise
```

**修复建议**：
1. 检查AI客户端的实际接口
2. 使用正确的调用方法
3. 添加详细的错误日志

  setProgressHistory(prev => {
    const newHistory = [...prev, js.progress!].slice(-3); // 保留最近3次进度
    
    // 检查是否连续3次进度未变化
    if (newHistory.length === 3 && 
        newHistory[0] === newHistory[1] && 
        newHistory[1] === newHistory[2] &&
        newHistory[0] < 100) {
      setShowStuckWarning(true);
    } else {
      setShowStuckWarning(false);
    }
    
    return newHistory;
  });
} else {
  // 重置检测状态

  setShowStuckWarning(false);
}
