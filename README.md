# ж”ҝеәңйў„еҶіз®—жЈҖжҹҘзі»з»ҹпјҲAIеўһејәзүҲпјү

## йЎ№зӣ®з®Җд»Ӣ

ж”ҝеәңйў„еҶіз®—жЈҖжҹҘзі»з»ҹжҳҜдёҖдёӘжҷәиғҪеҢ–зҡ„йў„еҶіз®—ж–ҮжЎЈеҲҶжһҗе·Ҙе…·пјҢж”ҜжҢҒдј з»ҹи§„еҲҷжЈҖжҹҘе’ҢAIжҷәиғҪеҲҶжһҗзҡ„еҸҢжЁЎејҸжЈҖжөӢпјҢеё®еҠ©жҸҗй«ҳйў„еҶіз®—е®ЎжҹҘзҡ„ж•ҲзҺҮе’ҢеҮҶзЎ®жҖ§гҖӮ

## рҹҡҖ ж ёеҝғзү№жҖ§

### еҸҢжЁЎејҸеҲҶжһҗеј•ж“Һ
- **AIжҷәиғҪеҲҶжһҗ**пјҡеҹәдәҺеӨ§иҜӯиЁҖжЁЎеһӢзҡ„жҷәиғҪй—®йўҳжЈҖжөӢпјҢж”ҜжҢҒеӨҡз§ҚAIжҸҗдҫӣе•Ҷ
- **и§„еҲҷеј•ж“ҺжЈҖжҹҘ**пјҡеҹәдәҺйў„е®ҡд№үи§„еҲҷзҡ„дј з»ҹжЈҖжөӢпјҲV3.3и§„еҲҷйӣҶпјү
- **еҸҢжЁЎејҸиһҚеҗҲ**пјҡеҗҢж—¶иҝҗиЎҢдёӨз§ҚжЁЎејҸе№¶жҷәиғҪеҗҲе№¶з»“жһң
- **еҶІзӘҒжЈҖжөӢ**пјҡиҮӘеҠЁиҜҶеҲ«е’ҢеӨ„зҗҶдёӨз§ҚжЁЎејҸй—ҙзҡ„еҲҶжӯ§
- **дёҖиҮҙжҖ§йӘҢиҜҒ**пјҡйӘҢиҜҒдёҚеҗҢжЈҖжөӢж–№ејҸзҡ„з»“жһңдёҖиҮҙжҖ§

### еӨҡAIжҸҗдҫӣе•Ҷж”ҜжҢҒ
- **OpenAIе…је®№жҺҘеҸЈ**пјҡж”ҜжҢҒOpenAI APIж јејҸзҡ„еҗ„з§ҚжңҚеҠЎ
- **жҷәи°ұAI (GLM)**пјҡж”ҜжҢҒжҷәи°ұжё…иЁҖзі»еҲ—жЁЎеһӢ
- **иұҶеҢ…AI (еӯ—иҠӮи·іеҠЁ)**пјҡж”ҜжҢҒиұҶеҢ…зі»еҲ—жЁЎеһӢ
- **иҮӘеҠЁж•…йҡңиҪ¬з§»**пјҡAIжңҚеҠЎејӮеёёж—¶иҮӘеҠЁеҲҮжҚўеҲ°и§„еҲҷеј•ж“Һ
- **иҙҹиҪҪеқҮиЎЎ**пјҡж”ҜжҢҒеӨҡжҸҗдҫӣе•Ҷй—ҙзҡ„жҷәиғҪи°ғеәҰ

### жҷәиғҪж–ҮжЎЈеӨ„зҗҶ
- **PDFж–ҮжЎЈи§Јжһҗ**пјҡж”ҜжҢҒеӨҚжқӮж јејҸзҡ„йў„еҶіз®—PDFж–ҮжЎЈ
- **OCRж–Үжң¬жҸҗеҸ–**пјҡжҷәиғҪиҜҶеҲ«иЎЁж јгҖҒеӣҫиЎЁдёӯзҡ„е…ій”®дҝЎжҒҜ
- **з»“жһ„еҢ–ж•°жҚ®жҸҗеҸ–**пјҡиҮӘеҠЁиҜҶеҲ«йў„з®—з§‘зӣ®гҖҒйҮ‘йўқгҖҒжҜ”дҫӢзӯүе…ій”®ж•°жҚ®
- **д№қиЎЁжЈҖжҹҘ**пјҡдё“й—Ёй’ҲеҜ№ж”ҝеәңйў„еҶіз®—д№қиЎЁзҡ„ж јејҸйӘҢиҜҒ

### зҺ°д»ЈеҢ–з”ЁжҲ·з•Ңйқў
- **е“Қеә”ејҸи®ҫи®Ў**пјҡж”ҜжҢҒжЎҢйқўе’Ңз§»еҠЁи®ҫеӨҮ
- **е®һж—¶еҲҶжһҗиҝӣеәҰ**пјҡеҸҜи§ҶеҢ–жҳҫзӨәеҲҶжһҗиҝӣеәҰе’ҢзҠ¶жҖҒ
- **еӨҡи§Ҷеӣҫеұ•зӨә**пјҡж ҮзӯҫйЎөгҖҒеҲ—иЎЁгҖҒеҚЎзүҮеӨҡз§Қз»“жһңеұ•зӨәж–№ејҸ
- **дәӨдә’ејҸз»“жһң**пјҡж”ҜжҢҒй—®йўҳзӯӣйҖүгҖҒжҗңзҙўгҖҒжҺ’еәҸзӯүж“ҚдҪң
- **еҶІзӘҒеҸҜи§ҶеҢ–**пјҡзӣҙи§Ӯеұ•зӨәAIе’Ңи§„еҲҷжЈҖжөӢзҡ„е·®ејӮ

## рҹ“Ұ е®үиЈ…йғЁзҪІ

### рҹҗҳ PostgreSQL ж•°жҚ®еә“и®ҫзҪ®пјҲеҸҜйҖүпјү

зі»з»ҹж”ҜжҢҒдҪҝз”Ё PostgreSQL еӯҳеӮЁеҲҶжһҗд»»еҠЎе’Ңз»“жһңгҖӮеҰӮжһңдёҚй…ҚзҪ®ж•°жҚ®еә“пјҢзі»з»ҹе°ҶдҪҝз”Ёж–Үд»¶зі»з»ҹеӯҳеӮЁгҖӮ

#### еҝ«йҖҹеҗҜеҠЁ

```bash

docker compose up -d

# 2. зӯүеҫ…е®№еҷЁеҒҘеә·зҠ¶жҖҒ
docker compose ps

# 3. еӨҚеҲ¶зҺҜеўғеҸҳйҮҸжЁЎжқҝ
cp .env.example .env

# 4. е®үиЈ… Python дҫқиө–пјҲеҢ…еҗ« asyncpgпјү
pip install -r requirements.txt

# 5. еҗҜеҠЁеҗҺз«ҜпјҲиҮӘеҠЁжү§иЎҢиҝҒз§»пјү
```

## рҹ”§ дҪҝз”ЁжҢҮеҚ—

### еҹәжң¬дҪҝз”ЁжөҒзЁӢ

1. **дёҠдј ж–ҮжЎЈ**пјҡйҖүжӢ©йў„еҶіз®—PDFж–Үд»¶иҝӣиЎҢдёҠдј 
2. **йҖүжӢ©жЁЎејҸ**пјҡ
   - **д»…AIеҲҶжһҗ**пјҡдҪҝз”ЁAIжЁЎеһӢиҝӣиЎҢжҷәиғҪжЈҖжөӢ
   - **д»…и§„еҲҷжЈҖжҹҘ**пјҡдҪҝз”Ёйў„е®ҡд№үи§„еҲҷиҝӣиЎҢжЈҖжөӢ
   - **еҸҢжЁЎејҸеҲҶжһҗ**пјҡеҗҢж—¶дҪҝз”ЁAIе’Ңи§„еҲҷиҝӣиЎҢжЈҖжөӢ
3. **жҹҘзңӢз»“жһң**пјҡзі»з»ҹе°ҶжҳҫзӨәжЈҖжөӢеҲ°зҡ„й—®йўҳе’Ңе»әи®®
4. **з»“жһңеҲҶжһҗ**пјҡеҸҜд»ҘеҲҮжҚўдёҚеҗҢи§ҶеӣҫжҹҘзңӢиҜҰз»Ҷз»“жһң

### еҸҢжЁЎејҸеҲҶжһҗиҜҰи§Ј

#### AIеҲҶжһҗжЁЎејҸ
- дҪҝз”ЁеӨ§иҜӯиЁҖжЁЎеһӢзҗҶи§Јж–ҮжЎЈеҶ…е®№
- иҜҶеҲ«еӨҚжқӮзҡ„иҜӯд№үй—®йўҳе’ҢйҖ»иҫ‘й”ҷиҜҜ
- жҸҗдҫӣжҷәиғҪеҢ–зҡ„й—®йўҳжҸҸиҝ°е’Ңе»әи®®
- йҖӮеҗҲеӨ„зҗҶйқһз»“жһ„еҢ–е’ҢеӨҚжқӮж јејҸзҡ„еҶ…е®№

#### и§„еҲҷжЈҖжҹҘжЁЎејҸ
- еҹәдәҺйў„е®ҡд№үзҡ„жЈҖжҹҘи§„еҲҷ
- еҝ«йҖҹиҜҶеҲ«ж јејҸгҖҒж•°еҖјиҢғеӣҙзӯүй—®йўҳ
- жҸҗдҫӣж ҮеҮҶеҢ–зҡ„й—®йўҳеҲҶзұ»е’ҢеӨ„зҗҶе»әи®®
- йҖӮеҗҲеӨ„зҗҶз»“жһ„еҢ–ж•°жҚ®е’Ңж ҮеҮҶж јејҸ

#### еҸҢжЁЎејҸиһҚеҗҲ
- **еҶІзӘҒжЈҖжөӢ**пјҡиҜҶеҲ«дёӨз§ҚжЁЎејҸз»“жһңзҡ„е·®ејӮ
- **дёҖиҮҙжҖ§йӘҢиҜҒ**пјҡйӘҢиҜҒзӣёеҗҢй—®йўҳзҡ„жЈҖжөӢдёҖиҮҙжҖ§
- **жҷәиғҪеҗҲе№¶**пјҡеҗҲе№¶дә’иЎҘзҡ„жЈҖжөӢз»“жһң
- **зҪ®дҝЎеәҰиҜ„дј°**пјҡеҹәдәҺеӨҡжЁЎејҸдёҖиҮҙжҖ§иҜ„дј°з»“жһңеҸҜйқ жҖ§
