# 预算规则设置建议（基于普陀区府办 2025 部门/单位样本）

## 1. 样本结论（用于定规则边界）
- 样本文件：
  - `X/上海市普陀区人民政府办公室25年预算.pdf`（22页，部门）
  - `X/上海市普陀区人民政府办公室单位25年预算.pdf`（21页，单位）
- 两份文件结构高度一致，均为“预算公开”而非“决算公开”。
- “九张预算表”均存在，空表（政府性基金、国有资本经营）均带规范注释。
- 表格单位大多为“元”，`“三公”经费和机关运行经费预算表`为“万元”，正文说明也用“万元”。

- 九张表：
  - T1 财务收支预算总表：P8

  - T7 国有资本经营预算支出功能分类预算表：P17（空表+注释）
  - T8 一般公共预算基本支出部门预算经济分类预算表：P18-P19

- 部门版额外“项目经费情况说明”：P22（单位版无此章）

## 3. 决算规则可复用/改造映射
- 可直接复用：
  - 结构完整性（必备表、必备章节）
  - 表内合计勾稽（合计=分项和）
  - 文本-表格一致性（金额与占比）
  - 空表说明、占位符、标点/格式检查
- 需改造：
  - 决算专属公式（如“结余分配 + 年末结转和结余 + 本年支出合计”）在预算表中不适用，改为预算总表与预算支出表的对应关系。
  - 年份规则不能简单“只允许当年”，预算文档会大量出现上一年（同比、资产截止日）。
  - 单位换算需支持“表格元 + 正文万元”的跨口径比对。

## 4. 建议的预算勾稽主链（优先级 P0）
- C001: `T1.收入总计 = T1.支出总计`
- C002: `T3.合计 = T3.基本支出 + T3.项目支出`
- C003: `T8.合计 = T8.人员经费 + T8.公用经费`
- C004: `T3.基本支出 = T8.合计`
- C005: `T1.收入总计 = T4.收入总计`
- C006: `T1.支出总计 = T4.支出总计(合计列)`
- C007: `T3(合计/基本/项目) = T5(合计/基本/项目)`
- C008: `T9.三公合计 = 因公出国(境) + 公务接待 + 公务用车购置及运行费小计`
- C009: `T9.公务用车购置及运行费小计 = 购置费 + 运行费`（空白按0处理）

## 5. 文本一致性链（优先级 P1）
- T001: 预算编制说明中的“收入预算、支出预算、财政拨款支出预算”对齐 T1/T4（支持元↔万元换算）。
- T002: “三公经费预算情况说明”对齐 T9 三公各项。
- T003: “机关运行经费预算”对齐 T9 机关运行经费。
- T004: 空表注释语义一致性：
  - 表为空且注释缺失 -> 报错
  - 表非空但注释写“无相关预算/本表为空” -> 报错

## 6. 规范性与低误报策略（优先级 P2）
- 年份规则：
  - 从封面提取报告年（如 2025）。
  - 允许 `{报告年, 报告年-1}`；出现 `报告年+1` 或 `报告年-2及以前`预警。
- 占位符规则：
  - `XX/xxx/待填写/请填写/(待)/(略)/[待]/[略]/{单位}/{部门}`。
- 标点/格式：
  - 重复标点、混合标点、明显错位单位口径。
- 单位换算：
  - 同一校验链路统一换算到“万元”，容差建议 `0.05万元`（=500元）。
